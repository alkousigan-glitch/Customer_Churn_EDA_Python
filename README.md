# Customer Churn Analysis — EDA with Python

## Problem Statement
A telecom company is experiencing a 15.6% churn rate.
This project identifies the key drivers of churn and
provides business recommendations to reduce it.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

## Key Findings
- 82% of customers who complained ended up churning
- 47.5% of inactive customers churned
- Churned customers used the service 3 to 5 times less
- Total revenue lost to churn is 27,784

## Dataset
Telecom Customer Churn Dataset — 2,851 rows, 13 features
